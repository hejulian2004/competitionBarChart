数据格式与使用说明
数据格式：第一行为表头；第一列是时间；后续每列是一个参赛对象。空白值按 0 处理；支持正数、0 和负数。 例如：年份,Python,Java,Kotlin。 Excel 文件默认读取第一个工作表。拖动“播放速度”滑块可实时调整速度，拖动“播放进度”可跳转到任意时间点。 横轴默认使用“平滑动态对称”：范围随数据平滑变化，但 0 刻度固定在中央，正负值保持相同比例； “平滑动态分侧”同样固定 0，但正负两侧独立缩放，更充分利用空间，不适合用柱长直接比较正负绝对值。 “自由动态”会移动 0 刻度。数值尺度默认为自动：当最大绝对值与最小非零绝对值相差超过约 1 万倍时， 自动启用支持负数和 0 的 Symlog 尺度。动态范围会随每帧数据缩放，可能造成抖动。 视频导出采用 Canvas 直接逐帧绘制和离线编码；默认优先生成适合 PowerPoint 的 H.264 MP4， 不支持时自动回退到 WebM。GIF 同样使用 Canvas 直接绘制，并为整段动画建立固定全局调色板。 弹幕会在网页、PNG、GIF 和视频中使用统一的独立卡片布局，并与柱形图保留安全间距。 文件仅在你的浏览器本地解析，不会由此页面上传到服务器。
